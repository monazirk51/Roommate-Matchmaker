# 🏠 Roommate Matchmaker

An AI-powered roommate matching system that helps students find compatible roommates based on their lifestyle, habits, and personal preferences. Instead of assigning roommates randomly, the system analyzes user data and recommends the most compatible matches to improve hostel living experiences.

---

## 📌 Project Overview

Finding a suitable roommate is one of the biggest challenges for students relocating to hostels or shared accommodations. The Roommate Matchmaker uses Machine Learning techniques to predict compatibility between users by analyzing their behavioral and lifestyle preferences.

The system collects user information through a questionnaire and recommends roommates with the highest compatibility score.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/roommate-matchmaker.git
```

Move into the project folder

```bash
cd roommate-matchmaker
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / Mac**

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```text
http://127.0.0.1:5000
```
